# CareerPilot AI

AI-powered career assistant for students and job seekers.


CareerPilot AI uses Azure AI to perform multi-step career reasoning by:

1. Understanding resume content
2. Comparing candidate skills with target roles
3. Identifying gaps
4. Creating personalized roadmaps
5. Generating interview preparation plans

## Features

- Resume Review
- Interview Preparation
- Skill Gap Analysis
- Resume Job Match
- Career Roadmap Generator
- Career Chat

## Tech Stack

- Python
- Streamlit
- Ollama
- Llama 3.2

## Installation

pip install -r requirements.txt

streamlit run app.py
